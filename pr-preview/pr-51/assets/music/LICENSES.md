## Processing

All tracks were processed for web use:
- Trimmed to ~60 second loops
- Converted to mono, 96kbps MP3
- Fade-out applied at end for seamless looping
- Metadata stripped for smaller file size

## Aggregation Source

Tracks were sourced from the [SoundSafari/CC0-1.0-Music](https://github.com/SoundSafari/CC0-1.0-Music) collection,
which aggregates public domain music from various free music websites.
